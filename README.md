# Itinerary Planner

Itinerary Planner is a Laravel-based web application that helps you organize trips and activities.

## Development

Install dependencies and run the development server:

```bash
composer install
npm install
npm run dev
php artisan serve
```

## Testing

Run the full test suite with:

```bash
composer test
```
